The scrollbar utility styles scrollbars with theme colors. Scrollbar styling is **opt-in**, omit the attribute to keep native browser scrollbars

## Import

```css
@import '@webtui/css/utils/scrollbar.css';
```

## Global Toggle

Set `data-webtui-scrollbar` on the `<html>` tag (or any container) to style all scrollable descendants

```html
<html data-webtui-scrollbar="themed"></html>
<html data-webtui-scrollbar="hidden"></html>
```

| Value    | Effect                                        |
| -------- | --------------------------------------------- |
| `themed` | Thin scrollbars using theme colors            |
| `hidden` | Hide scrollbars while keeping scroll behavior |
| _(omit)_ | Native browser scrollbars                     |

### Customizing

**Themed scrollbars read `--foreground1`, `--background2`, and `--foreground0`. Override those on a container to restyle scrollbars without touching the utility:**

```css
#my-scroll-area {
    --foreground1: var(--background3);
    --background2: var(--background1);
}
```
